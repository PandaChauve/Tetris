When Will You Lose

WWYL is a game about survival in the ruthless world of falling blocks.

The concept is quite simple, you can swap blocks horizontaly, when 3 or more blocks of a same color are aligned (verticaly or horizontaly) they're destroyed.

You will lose !

Score & points

The basic unit is the line, to create one you need 3 blocks aligned.

Each line is worth one point. Each additional block on a line is 1 point (a line of 5 blocks is 3 points).

If you create a cross or a corner (an horizontal and a vertical line combined) they count as a single line and the block used to join them is counted twice. So it's a least 5 points.

Finally if you do multiple lines at the same time, you'll get the sum of each individual score multiplied by the number of lines. The same effect is applied if you create lines with falling blocks.

Single player modes

Arcade

Classic

The classic mode is a survival game in a 6X10 grid. Blocks will fall from the top and the grid is going up at an increasing speed. The more you play, the harder it gets !

Tips : at the beginning the game is slow, it's easier to create big combinations. You should use the speed key to insure a sufficient block pool for your combos. When the game start to be too fast for you, focus on destroying small blocks, the more you play the more you score !

Sandbox

The ground won't move, the blocks wont fall. Take your time to train your combos. You can still use the "Speed up" key to add new lines to the grid.

Campaign

Challenges

No falling blocks, no moving ground, just some objectives and constraints. Reach a score, destroy all the blocks, destroy only one color, keep only a color,... You'll need your brain.

Arcade

Do you remember the arcade modes (classic, wide, crazy) ? It's that with a chance to win ! Stay in game long enough !

Time attack

You have a limited time to score a certain amount of points. Some grids are randomly generated, other are not !

Tetris attack

You have a limited amount of swaps to destroy all the blocks. This is a clone of Tetris Attack's puzzle mode, 60 levels of increasing difficulty. All the levels can be completed with 5 sec between each swap, you don't need to try skillful moves.

Multiplayer modes

You can't play them on mobile device.

Split screen

Same rules as the single player mode "Classic" with 2 players. Each time you score 3 points, some blocks will fall in your partners grid. Show that you're the best

Coop

Same rules as the single player mode "Wide" but you can play with a friend.
